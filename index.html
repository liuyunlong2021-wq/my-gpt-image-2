🔌 使用 GPT Image 2 API

一次 API 调用即可完成生成和编辑 — 兼容 OpenAI 标准格式。

⚡快速开始

使用单条 curl 命令通过 GPT Image 2 API 生成图片。

curl --request POST \ --url https://api.evolink.ai/v1/images/generations \ --header 'Authorization: Bearer YOUR_API_KEY' \ --header 'Content-Type: application/json' \ --data '{ "model": "gpt-image-2", "prompt": "A beautiful sunset over the ocean, photorealistic, 8K" }'

🔧安装 Skill

一行命令安装 GPT-Image-2 Gen Skill，无缝集成。

npx evolink-gpt-image -y
