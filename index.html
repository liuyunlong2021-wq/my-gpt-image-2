🔌 Use GPT Image 2 API

One API call for both generation and editing — works with OpenAI's standard format.

⚡Quick Start

Generate an image with a single curl command using the GPT Image 2 API.

curl --request POST \ --url https://api.evolink.ai/v1/images/generations \ --header 'Authorization: Bearer YOUR_API_KEY' \ --header 'Content-Type: application/json' \ --data '{ "model": "gpt-image-2", "prompt": "A beautiful sunset over the ocean, photorealistic, 8K" }'

🔧Install the Skill

One-line install of the GPT-Image-2 Gen Skill for seamless integration.

npx evolink-gpt-image -y
